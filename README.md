# util

A mono-repo of stable, generic utility modules for Go projects.

Each subdirectory is an independent Go module with its own `go.mod`. Modules are tagged as
`<name>/v<semver>` (e.g. `slogger/v0.1.0`).

This library supersedes [github.com/mrmxf/clog](https://github.com/mrmxf/clog).
